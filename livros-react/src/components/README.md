# Acervo de Livros

Aplicação frontend em React que consome uma API REST de livros.

## Tecnologias usadas
- React
- Vite
- Axios
- Node.js
- Express
- localStorage

## Como rodar

### API (backend)
```bash
cd api-livros
npm install
npm start
```

### Frontend
```bash
cd livros-react
npm install
npm run dev
```

## Funcionalidades
- Listar livros
- Cadastrar livro
- Editar livro
- Excluir livro
- Filtrar por gênero
- Filtro salvo no localStorage
